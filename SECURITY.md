# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ClawGraph, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email: **security@clawgraph.ai**

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

- ClawGraph CLI and library code
- Cypher injection via LLM-generated queries
- Configuration file handling
- Dependency vulnerabilities

## Out of Scope

- Vulnerabilities in third-party LLM providers
- Issues in the Kùzu database engine itself (report to kuzudb.com)
